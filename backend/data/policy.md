# 电商平台售后与退换货政策

## 1. 退换货期限
- **七天无理由退货**：自买家签收商品之日起7天内，商品完好、附件齐全且不影响二次销售的前提下，支持七天无理由退货。
- **十五天换货**：自买家签收商品之日起15天内，若商品出现国家三包规定的性能故障或质量问题，买家可申请免费换货。

## 2. 运费规则
- **七天无理由退货**：因买家主观原因（如不喜欢、拍错、不合适等）导致的退换货，**来回运费均由买家承担**。
- **质量问题退换货**：若商品存在破损、发错货或功能性故障，**退换货产生的运费由商家承担**。买家需先垫付退回运费，商家收到退件确认后通过系统补偿。
- **包邮商品退货**：若商品本身为包邮商品，买家退货时仅需承担退回运费；若为非包邮商品，买家需承担发货和退回的双程运费。

## 3. 特殊商品退换货说明
以下商品不支持七天无理由退货：
1. 定制类商品（如刻字、特殊尺寸等）。
2. 鲜活易腐类商品（如生鲜食品、花卉等）。
3. 在线下载或者买家已拆封的音像制品、计算机软件等数字化商品。
4. 交付的报纸、期刊。
5. 拆封后易影响人身安全或者生命健康的商品，或者拆封后易导致商品品质发生改变的商品（如贴身内衣、食品等）。

## 4. 退款时效
- 商家在收到退货包裹并验收无误后，将在 **3个工作日内** 完成退款操作。
- 退款金额将原路返回至买家的支付账户，具体到账时间视银行或支付平台（如微信、支付宝）的处理速度而定，一般为 1-5 个工作日。
